# 數位行銷 Digital Marketing

> 數據驅動的行銷策略與執行

## 適用場景

- 制定行銷策略與預算分配
- 內容行銷規劃與執行
- SEO 優化與關鍵字研究
- 廣告投放與成效優化
- 社群媒體經營

## 行銷漏斗

```
┌─────────────────────────────────────────────────────────────────┐
│  AARRR 海盜指標                                                 │
│                                                                 │
│  Acquisition  → Activation → Retention → Revenue → Referral   │
│    獲取用戶      激活用戶     留存用戶    變現      推薦        │
│                                                                 │
│  指標：                                                         │
│  流量來源       註冊率       DAU/MAU    ARPU      NPS          │
│  CAC           首次使用     留存率     LTV       推薦率        │
└─────────────────────────────────────────────────────────────────┘
```

## 核心知識

### 行銷渠道

| 渠道 | 適用 | 成本 | 效果週期 |
|------|------|------|----------|
| **SEO** | 長期流量 | 低 | 3-6 個月 |
| **SEM/PPC** | 即時轉換 | 高 | 即時 |
| **社群媒體** | 品牌建立 | 中 | 1-3 個月 |
| **內容行銷** | 信任建立 | 中 | 3-6 個月 |
| **Email** | 留存轉換 | 低 | 即時 |

### 關鍵指標

| 指標 | 說明 | 計算方式 |
|------|------|----------|
| **CAC** | 獲客成本 | 行銷費用 / 新客數 |
| **LTV** | 客戶終身價值 | ARPU × 生命週期 |
| **ROAS** | 廣告投報率 | 廣告收入 / 廣告成本 |
| **CTR** | 點擊率 | 點擊數 / 曝光數 |
| **CVR** | 轉換率 | 轉換數 / 點擊數 |

## 最佳實踐

1. **數據先行** - 所有決策基於數據，不憑感覺
2. **A/B 測試** - 持續測試優化，小步快跑
3. **用戶導向** - 了解目標用戶的痛點和需求
4. **整合行銷** - 多渠道協同，一致的品牌訊息
5. **ROI 思維** - 關注投資報酬率，砍掉低效渠道

## 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| ❌ 追求虛榮指標（粉絲數） | ✅ 關注轉換指標（營收） |
| ❌ 全渠道撒網 | ✅ 專注高效渠道深耕 |
| ❌ 一次大改版 | ✅ 持續 A/B 測試迭代 |
| ❌ 只看短期效果 | ✅ 平衡短期和長期 |

## SEO 優化指南

### On-Page SEO 檢查清單

```markdown
## 頁面 SEO 優化

### 標題標籤 (Title Tag)
- [ ] 包含主要關鍵字
- [ ] 長度 50-60 字元
- [ ] 吸引人點擊

### Meta Description
- [ ] 包含關鍵字
- [ ] 長度 150-160 字元
- [ ] 包含行動呼籲

### 內容優化
- [ ] H1 只有一個，包含關鍵字
- [ ] H2-H6 結構清晰
- [ ] 關鍵字密度 1-2%
- [ ] 內部連結 3-5 個
- [ ] 圖片有 alt 標籤

### 技術面
- [ ] 網頁載入 < 3 秒
- [ ] 行動裝置友善
- [ ] HTTPS 安全連線
- [ ] 結構化資料 (Schema)
```

### 關鍵字研究流程

```
種子關鍵字 → 擴展長尾 → 評估競爭度 → 意圖分析 → 內容規劃
     ↓           ↓            ↓           ↓          ↓
   「行銷」  「數位行銷教學」  KD < 30    資訊型    教學文章
                            搜尋量>100  交易型    產品頁面
```

## 廣告投放策略

### Facebook/Meta 廣告架構

```
Campaign（目標）
    └── Ad Set（受眾、預算、版位）
            └── Ad（創意、文案、CTA）

最佳實踐：
• 每個 Ad Set 放 3-6 個 Ad 變體
• 預算分配：測試 20% / 擴展 80%
• 先廣泛受眾測試，再縮小精準
```

### Google Ads 類型選擇

| 類型 | 適用場景 | 優勢 |
|------|----------|------|
| **搜尋廣告** | 主動搜尋需求 | 意圖明確 |
| **多媒體廣告** | 品牌曝光 | 成本較低 |
| **購物廣告** | 電商產品 | 視覺吸引 |
| **影片廣告** | 品牌故事 | 高互動 |

## Email 行銷最佳實踐

### Email 類型

| 類型 | 目的 | 頻率 |
|------|------|------|
| **歡迎信** | 建立關係 | 註冊後立即 |
| **電子報** | 持續互動 | 週/雙週 |

### 開信率優化

```markdown
## 提升開信率

主旨行技巧：
✅ 製造好奇：「你錯過了這個...」
✅ 數字具體：「3 個方法提升轉換率 50%」
✅ 個人化：「[姓名]，專屬優惠給你」
✅ 急迫感：「限時 24 小時」

避免：
❌ 全大寫
❌ 過多驚嘆號
❌ 垃圾信觸發詞（免費、賺錢）
```

## 行銷預算分配

### 預算分配框架

```
┌─────────────────────────────────────────────────────────────────┐
│  70-20-10 法則                                                   │
│                                                                 │
│  70% - 已驗證有效的渠道（持續優化）                             │
│  20% - 新興但有潛力的渠道（擴展測試）                           │
│  10% - 實驗性渠道（探索創新）                                   │
│                                                                 │
│  範例（月預算 100 萬）：                                         │
│  • Google Ads：50 萬（已驗證）                                  │
│  • Facebook：20 萬（已驗證）                                    │
│  • TikTok：20 萬（新興）                                        │
│  • Podcast 贊助：10 萬（實驗）                                  │
└─────────────────────────────────────────────────────────────────┘
```
