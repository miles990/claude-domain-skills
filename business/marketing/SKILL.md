# 數位行銷 Digital Marketing

> 數據驅動的行銷策略與執行

## 適用場景

- 制定行銷策略與預算分配
- 內容行銷規劃與執行
- SEO 優化與關鍵字研究
- 廣告投放與成效優化
- 社群媒體經營

## 行銷漏斗

```
┌─────────────────────────────────────────────────────────────────┐
│  AARRR 海盜指標                                                 │
│                                                                 │
│  Acquisition  → Activation → Retention → Revenue → Referral   │
│    獲取用戶      激活用戶     留存用戶    變現      推薦        │
│                                                                 │
│  指標：                                                         │
│  流量來源       註冊率       DAU/MAU    ARPU      NPS          │
│  CAC           首次使用     留存率     LTV       推薦率        │
└─────────────────────────────────────────────────────────────────┘
```

## 核心知識

### 行銷渠道

| 渠道 | 適用 | 成本 | 效果週期 |
|------|------|------|----------|
| **SEO** | 長期流量 | 低 | 3-6 個月 |
| **SEM/PPC** | 即時轉換 | 高 | 即時 |
| **社群媒體** | 品牌建立 | 中 | 1-3 個月 |
| **內容行銷** | 信任建立 | 中 | 3-6 個月 |
| **Email** | 留存轉換 | 低 | 即時 |

### 關鍵指標

| 指標 | 說明 | 計算方式 |
|------|------|----------|
| **CAC** | 獲客成本 | 行銷費用 / 新客數 |
| **LTV** | 客戶終身價值 | ARPU × 生命週期 |
| **ROAS** | 廣告投報率 | 廣告收入 / 廣告成本 |
| **CTR** | 點擊率 | 點擊數 / 曝光數 |
| **CVR** | 轉換率 | 轉換數 / 點擊數 |

## 最佳實踐

1. **數據先行** - 所有決策基於數據，不憑感覺
2. **A/B 測試** - 持續測試優化，小步快跑
3. **用戶導向** - 了解目標用戶的痛點和需求
4. **整合行銷** - 多渠道協同，一致的品牌訊息
5. **ROI 思維** - 關注投資報酬率，砍掉低效渠道

## 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| ❌ 追求虛榮指標（粉絲數） | ✅ 關注轉換指標（營收） |
| ❌ 全渠道撒網 | ✅ 專注高效渠道深耕 |
| ❌ 一次大改版 | ✅ 持續 A/B 測試迭代 |
| ❌ 只看短期效果 | ✅ 平衡短期和長期 |

- [ ] 包含關鍵字
- [ ] 長度 150-160 字元

- [ ] 網頁載入 < 3 秒

## 廣告投放策略

### Facebook/Meta 廣告架構

```
Campaign（目標）
    └── Ad Set（受眾、預算、版位）
            └── Ad（創意、文案、CTA）

最佳實踐：
• 每個 Ad Set 放 3-6 個 Ad 變體
• 預算分配：測試 20% / 擴展 80%
• 先廣泛受眾測試，再縮小精準
```

### 用戶心理學

#### 說服原則（Cialdini's 6 Principles）

| 原則 | 應用 | 行銷範例 |
|------|------|----------|
| **互惠** | 先給予價值 | 免費試用、電子書 |
| **稀缺** | 限量限時 | 「僅剩 3 個名額」 |
| **權威** | 專家背書 | 專家推薦、認證標章 |
| **一致性** | 先小後大 | 免費試用 → 付費 |
| **喜好** | 情感連結 | 品牌故事、代言人 |
| **社會認同** | 從眾心理 | 客戶見證、使用人數 |

```
┌─────────────────────────────────────────────────────────────────┐

### 社群互動公式

```
互動率 = (按讚 + 留言 + 分享 + 儲存) / 觸及人數 × 100%

基準值：
• Facebook：1-3%（好）、3-6%（優秀）
• Instagram：3-6%（好）、6%+（優秀）
• LinkedIn：2-4%（好）、4%+（優秀）

提升互動技巧：
1. 提問結尾（「你怎麼看？」）
2. 投票/選擇題
3. 使用表情符號（增 25% 互動）
4. 回覆每則留言（前 1 小時內）
5. 標註相關帳號
```

## Influencer 行銷

### KOL 分級與合作策略

| 等級 | 粉絲數 | 特點 | 合作方式 | 預算 |
|------|--------|------|----------|------|
| **Mega** | 100萬+ | 高曝光、低互動 | 品牌大使 | $$$$ |
| **Macro** | 10-100萬 | 專業領域影響力 | 付費合作 | $$$ |
| **Micro** | 1-10萬 | 高互動、社群信任 | 產品置換/付費 | $$ |
| **Nano** | 1萬以下 | 極高互動、真實 | 產品置換 | $ |

### 合作評估公式

```
影響力分數 = (互動率 × 粉絲數 × 相關性) / 價格

評估清單：
□ 粉絲真實性（假粉檢測工具）
□ 互動品質（留言是否真實）
□ 過往合作案例
□ 受眾重疊度
□ 品牌安全性（過往爭議）
```

    │   開信率：45%，轉換率：10%
    │
    ├─ 24 小時後：社會認同（產品評價）
    │   開信率：30%，轉換率：5%
    │
    └─ 72 小時後：限時優惠（10% 折扣）
        開信率：25%，轉換率：8%

總挽回率預估：15-20%
```
