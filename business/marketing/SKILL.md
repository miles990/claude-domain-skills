# 數位行銷 Digital Marketing

> 數據驅動的行銷策略與執行

## 適用場景

- 制定行銷策略與預算分配
- 內容行銷規劃與執行
- SEO 優化與關鍵字研究
- 廣告投放與成效優化
- 社群媒體經營

## 行銷漏斗

```
┌─────────────────────────────────────────────────────────────────┐
│  AARRR 海盜指標                                                 │
│                                                                 │
│  Acquisition  → Activation → Retention → Revenue → Referral   │
│    獲取用戶      激活用戶     留存用戶    變現      推薦        │
│                                                                 │
│  指標：                                                         │
│  流量來源       註冊率       DAU/MAU    ARPU      NPS          │
│  CAC           首次使用     留存率     LTV       推薦率        │
└─────────────────────────────────────────────────────────────────┘
```

## 核心知識

### 行銷渠道

| 渠道 | 適用 | 成本 | 效果週期 |
|------|------|------|----------|
| **SEO** | 長期流量 | 低 | 3-6 個月 |
| **SEM/PPC** | 即時轉換 | 高 | 即時 |
| **社群媒體** | 品牌建立 | 中 | 1-3 個月 |
| **內容行銷** | 信任建立 | 中 | 3-6 個月 |
| **Email** | 留存轉換 | 低 | 即時 |

### 關鍵指標

| 指標 | 說明 | 計算方式 |
|------|------|----------|
| **CAC** | 獲客成本 | 行銷費用 / 新客數 |
| **LTV** | 客戶終身價值 | ARPU × 生命週期 |
| **ROAS** | 廣告投報率 | 廣告收入 / 廣告成本 |
| **CTR** | 點擊率 | 點擊數 / 曝光數 |
| **CVR** | 轉換率 | 轉換數 / 點擊數 |

## 最佳實踐

1. **數據先行** - 所有決策基於數據，不憑感覺
2. **A/B 測試** - 持續測試優化，小步快跑
3. **用戶導向** - 了解目標用戶的痛點和需求
4. **整合行銷** - 多渠道協同，一致的品牌訊息
5. **ROI 思維** - 關注投資報酬率，砍掉低效渠道

## 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| ❌ 追求虛榮指標（粉絲數） | ✅ 關注轉換指標（營收） |
| ❌ 全渠道撒網 | ✅ 專注高效渠道深耕 |
| ❌ 一次大改版 | ✅ 持續 A/B 測試迭代 |
| ❌ 只看短期效果 | ✅ 平衡短期和長期 |
