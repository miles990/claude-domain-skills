# 故事創作 Storytelling

> 打造引人入勝的故事，讓讀者/觀眾無法自拔

## 適用場景

- 小說/短篇故事創作
- 漫畫腳本與分鏡
- 影片/短片劇本
- 遊戲敘事設計
- 品牌故事行銷

## 故事結構

### 三幕劇結構

```

│       │                   │                   │                 │
│  ─────●───────────────────●───────────────────●─────→           │
│    Inciting          Midpoint            Climax                │
│    Event             轉折點              高潮                   │
│                                                                 │

```

### 英雄旅程 (Hero's Journey)

```markdown
## 12 階段英雄旅程

### 出發 (Departure)
1. **平凡世界** - 日常生活
2. **冒險召喚** - 觸發事件
3. **拒絕召喚** - 猶豫恐懼
4. **遇見導師** - 獲得指引
5. **跨越門檻** - 進入新世界

### 啟蒙 (Initiation)
6. **考驗與盟友** - 結交夥伴，面對挑戰
7. **逼近洞穴** - 接近核心危機
8. **苦難折磨** - 最大的考驗
9. **獎賞** - 獲得寶物/能力

### 回歸 (Return)
10. **回程之路** - 返回途中
11. **復活** - 最終考驗
12. **帶著寶藏歸來** - 改變後的新生活
```

### 故事脊椎 (Story Spine)

```markdown
## Pixar Story Spine

Once upon a time there was ___. (從前有...)
Every day, ___. (每天...)
One day ___. (有一天...)
Because of that, ___. (因此...)
Because of that, ___. (因此...)
Because of that, ___. (因此...)
Until finally ___. (直到最後...)
And ever since then ___. (從此以後...)

範例：
從前有一個害羞的機器人瓦力。
每天他獨自清理垃圾。
有一天，一個叫伊芙的機器人來了。
因此，瓦力愛上了她並跟隨她到太空船。
因此，他意外啟動了返回地球的計劃。
因此，人類終於回到地球。
直到最後，瓦力和伊芙在一起了。
從此以後，地球開始復甦。
```

## 對話寫作

### 好對話的特徵

| 特徵 | 說明 | 範例 |
|------|------|------|
| **潛台詞** | 言外之意 | 「天氣真好」(想結束話題) |
| **衝突** | 角色目標對立 | A 想離開，B 想挽留 |
| **個性化** | 反映角色特質 | 教授說話 vs 小孩說話 |
| **推進劇情** | 每句話有目的 | 透露資訊或改變關係 |

### 對話技巧

```markdown
## 對話寫作原則

1. **少即是多**
   ❌ 「我今天早上去了超市，買了牛奶和麵包。」
   ✅ 「冰箱空了。」

2. **行動代替說明**
   ❌ 「我很生氣！」他生氣地說。
   ✅ 他把門摔上。

3. **打斷與沉默**
   - 打斷 = 強勢/急切
   - 沉默 = 思考/抗拒

4. **間接回答**
   「你愛我嗎？」
   ❌ 「是的，我愛你。」
   ✅ 「你的咖啡涼了。」
```

## 漫畫腳本

### 分鏡設計

```
┌─────────────────────────────────────────────────────────────────┐
│  漫畫分鏡基本格式                                               │
│                                                                 │
│  頁數：P.XX                                                     │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ 格 1 (大)                                                │   │
│  │ 景別：遠景                                               │   │
│  │ 描述：城市天際線，夕陽西下                               │   │
│  │ 對話：（無）                                             │   │
│  │ 效果：建立氛圍                                           │   │
│  ├─────────────────────┬───────────────────────────────────┤   │
│  │ 格 2                 │ 格 3                              │   │
│  │ 景別：中景           │ 景別：特寫                        │   │
│  │ 描述：主角走在街上   │ 描述：主角的眼睛                  │   │
│  │ 對話：...            │ 對話：「終於到了。」              │   │
│  └─────────────────────┴───────────────────────────────────┘   │
│                                                                 │
│  景別參考：                                                     │
│  遠景 → 建立場景                                               │
│  全景 → 展示角色全身                                           │
│  中景 → 對話場景                                               │
│  特寫 → 強調情感                                               │
│  大特寫 → 物件或表情細節                                       │
└─────────────────────────────────────────────────────────────────┘
```

### 漫畫節奏

```markdown
## 節奏控制

### 加速（緊張/動作）
- 格子變小變多
- 斜線構圖
- 速度線
- 少對話

### 減速（情感/思考）
- 格子變大
- 留白空間
- 特寫表情
- 內心獨白

### 翻頁驚喜
- 重要畫面放在翻頁後
- 懸念放在右頁最後一格
```

## 影片腳本

### 腳本格式

```markdown
## 短片腳本模板

場景 1 - 內景/咖啡廳/日

[描述]
陽光從落地窗灑入。小美（25歲）獨自坐在角落，
盯著手機螢幕。

[動作]
小美滑動手機，表情從期待變成失望。

[對話]
小美：（自言自語）又已讀不回...

[過場]
手機螢幕特寫，顯示「對方正在輸入...」

---

場景 2 - 外景/街道/日

...
```

### 影片結構

| 類型 | 時長 | 結構 |

| **短影片** | 15-60s | Hook → 內容 → CTA |

## 世界觀建構

### 世界觀要素

```markdown
## Worldbuilding Checklist

### 基礎設定
- [ ] 時代背景（過去/現代/未來）
- [ ] 地理環境
- [ ] 科技水平
- [ ] 魔法/超能力系統（如有）

### 社會結構
- [ ] 政治體制
- [ ] 經濟系統
- [ ] 社會階層
- [ ] 文化習俗

### 規則與限制
- [ ] 什麼是可能的？
- [ ] 什麼是禁止的？
- [ ] 違反規則的後果？

### 歷史
- [ ] 重大歷史事件
- [ ] 如何影響現在？
```

## 常見問題

| 問題 | 解決方案 |
|------|----------|
| 開頭太慢 | 從動作/衝突開始 |
| 中段無聊 | 加入轉折和障礙 |
| 角色扁平 | 加入缺陷和矛盾 |
| 結局無力 | 呼應開頭，角色弧線完整 |
| 對話生硬 | 加入潛台詞，減少直述 |
