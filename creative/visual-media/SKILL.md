## 攝影基礎

### 曝光三角

```
┌─────────────────────────────────────────────────────────────────┐
│  曝光三角 Exposure Triangle                                     │
│                                                                 │
│              光圈 Aperture                                      │
│                 /\                                              │
│                /  \                                             │
│               /    \                                            │
│              /      \                                           │
│             /        \                                          │
│            /          \                                         │
│           /            \                                        │
│          /______________\                                       │
│    快門 Shutter    ISO 感光度                                   │
│                                                                 │
│  光圈 (f/1.4 - f/22)                                           │
│  ├─ 大光圈 (小數字)：淺景深，背景模糊                          │
│  └─ 小光圈 (大數字)：深景深，全畫面清晰                        │
│                                                                 │

│  ├─ 低 ISO：畫質細膩                                           │
│  └─ 高 ISO：可用於暗光，但有噪點                               │
└─────────────────────────────────────────────────────────────────┘
```

### 構圖法則

```markdown
## 基本構圖

### 三分法
- 將畫面分成 3×3 格
- 主體放在交叉點
- 地平線放在上/下三分線

### 引導線
- 利用線條引導視線
- 道路、河流、欄杆
- 指向主體

### 框架構圖
- 用前景元素框住主體
- 門框、窗戶、樹枝
- 增加層次感

### 對稱與平衡
- 完美對稱 = 穩定、莊嚴
- 不對稱 = 動態、張力
- 視覺重量平衡

### 負空間
- 留白增加焦點
- 給主體「呼吸空間」
- 極簡風格
```

### 光線運用

| 光線類型 | 特點 | 適合 |
|----------|------|------|
| **順光** | 平淡、少陰影 | 記錄、證件 |
| **側光** | 立體、有陰影 | 人像、建築 |
| **逆光** | 剪影、光暈 | 氛圍、藝術 |
| **頂光** | 強烈陰影 | 正午避免 |
| **黃金時刻** | 柔和暖色 | 風景、人像 |
| **藍調時刻** | 冷色調、神秘 | 城市、夜景 |

### 運鏡技巧

| 運鏡 | 效果 | 使用場景 |
|------|------|----------|
| **推 (Push in)** | 強調、逼近 | 揭示重點 |
| **拉 (Pull out)** | 揭示全貌 | 結尾、轉場 |
| **搖 (Pan)** | 水平移動 | 跟隨、環境介紹 |
| **升降 (Tilt)** | 垂直移動 | 展示高度 |
| **跟 (Track)** | 跟隨主體 | 行走、追逐 |
| **環繞 (Orbit)** | 360度繞行 | 戲劇性時刻 |
| **手持 (Handheld)** | 臨場感 | 紀錄片、緊張 |

### 景別應用

```markdown
## 景別 Shot Types

### 遠景 (Wide Shot)
- 建立場景
- 展示環境
- 角色渺小

### 全景 (Full Shot)
- 角色全身
- 動作清晰
- 服裝展示

### 中景 (Medium Shot)
- 腰部以上
- 對話場景

### 近景 (Close-up)
- 肩部以上
- 情感表達
- 重要對白

### 大特寫 (Extreme Close-up)
- 眼睛、手部細節
- 強烈情緒
- 物件特寫
```

## 動畫基礎

### 動畫 12 法則

```markdown
## Disney 12 Principles of Animation

1. **擠壓與拉伸** (Squash & Stretch)
   - 保持體積
   - 表現重量和彈性

2. **預備動作** (Anticipation)
   - 跳之前先蹲
   - 出拳前先收

3. **演出** (Staging)
   - 清晰傳達意圖
   - 引導觀眾視線

4. **連續動作與關鍵格** (Straight Ahead & Pose to Pose)
   - 連續：流暢、有機
   - 關鍵格：精準、可控

5. **跟隨與重疊** (Follow Through & Overlapping)
   - 頭髮、衣服的延遲
   - 增加自然感

6. **緩入緩出** (Slow In & Slow Out)
   - 動作開始和結束較慢
   - 中間較快

7. **弧線** (Arcs)
   - 自然動作是曲線
   - 避免機械感

8. **附屬動作** (Secondary Action)
   - 主動作之外的小動作
   - 豐富角色

9. **時間掌握** (Timing)
   - 格數決定速度
   - 影響重量感

10. **誇張** (Exaggeration)
    - 比現實更強烈
    - 突出特徵

11. **紮實繪圖** (Solid Drawing)
    - 理解立體結構
    - 重量和平衡

12. **吸引力** (Appeal)
    - 角色魅力
    - 讓人想看
```

## 後製剪輯

### 剪輯節奏

```markdown
## 剪輯原則

### 動接動
- 動作中切換鏡頭
- 維持流暢感

### 靜接靜
- 停頓時切換
- 對話場景常用

### 軸線規則
- 保持角色相對位置
- 避免跳軸

### J-Cut / L-Cut
- J-Cut：聲音先於畫面
- L-Cut：畫面先於聲音
- 增加流暢度
```

### 調色基礎

| 調整項目 | 作用 | 範例 |
|----------|------|------|
| 曝光 | 整體亮度 | 過暗補光 |
| 對比 | 明暗差距 | 增加立體 |
| 色溫 | 冷暖色調 | 暖色溫馨 |
| 色調 | 綠/洋紅 | 膚色校正 |
| 飽和度 | 色彩濃度 | 電影感降飽和 |
| 曲線 | 精細調整 | S曲線增對比 |
