## 投資理財

### 投資工具比較

| 工具 | 風險 | 報酬潛力 | 流動性 | 適合 |
|------|------|----------|--------|------|
| **定存** | 極低 | 1-2% | 高 | 緊急備用 |
| **債券/ETF** | 低 | 3-5% | 高 | 穩定配置 |
| **股票 ETF** | 中 | 7-10% | 高 | 長期投資 |
| **個股** | 高 | 不定 | 高 | 有研究能力 |
| **加密貨幣** | 極高 | 極高/極低 | 高 | 風險承受高 |
| **房地產** | 中 | 5-10%+ | 低 | 長期資產 |

### 資產配置原則

```markdown
## 理財金字塔

1. **緊急備用金** (6 個月生活費)
   └─ 高流動性，隨時可用

2. **保險保障**
   └─ 醫療、意外、壽險

3. **穩定投資** (50-70%)
   └─ ETF、債券、定期定額

4. **成長投資** (20-40%)
   └─ 個股、產業 ETF

5. **投機配置** (5-10%)
   └─ 加密貨幣、新興市場

## 簡單法則
- 100 - 年齡 = 股票比例
- 不懂的不投
- 分散配置，長期持有
```

### 加密貨幣入門

```markdown
## 加密貨幣基礎

### 主流幣種
| 幣種 | 類型 | 用途 |
|------|------|------|
| BTC | 價值儲存 | 數位黃金 |
| ETH | 智能合約平台 | DeFi、NFT |
| USDT/USDC | 穩定幣 | 錨定美元 |

### 投資策略
1. **DCA 定期定額** - 最安全的方式
2. **長期持有** - 不看短期波動
3. **嚴格止損** - 設定可承受虧損
4. **分散配置** - 不超過投資組合 10%

### 風險提醒
⚠️ 高波動，可能歸零
⚠️ 詐騙多，注意安全
⚠️ 稅務問題需留意
⚠️ 只投入可承受損失的金額
```

### 定價策略

```markdown
## 報價方式

### 時薪制
- 適合：不確定範圍的專案
- 缺點：收入有天花板

### 專案制
- 適合：明確範圍的專案
- 計算：(預估時數 × 時薪) × 1.5 緩衝

### 價值定價
- 適合：能量化價值的案子
- 計算：為客戶創造價值的 10-20%

## 報價心法
- 報價要有自信
- 說明價值，不只是價格
- 提供選項 (基本/標準/豪華)
- 寧願報高被砍，不要報低
```

## 避免陷阱

| 陷阱 | 提醒 |
|------|------|
| 快速致富 | 天下沒有白吃的午餐 |
| 高報酬低風險 | 不存在，小心詐騙 |
| 借錢投資 | 絕對不要 |
| 跟風投資 | 做好自己的研究 |
| 忽略稅務 | 副業收入要報稅 |
