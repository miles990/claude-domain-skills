## 目標設定

### 個人 OKR

```markdown
## 季度個人 OKR 模板

### Objective 1: [具體且有啟發性的目標]

KR1: [可量化的關鍵結果]
     目前：0% → 目標：100%

KR2: [可量化的關鍵結果]
     目前：0 → 目標：N

KR3: [可量化的關鍵結果]
     目前：X → 目標：Y

### 範例

Objective: 成為更好的公開演講者

KR1: 完成 3 場公開演講
KR2: 演講滿意度達 4.5/5
KR3: 每週練習 2 次，共 24 次
```

### SMART 目標

| 要素 | 說明 | 範例 |
|------|------|------|
| **S** Specific | 具體明確 | 「減重」→「減掉 5 公斤」 |
| **M** Measurable | 可衡量 | 可以追蹤進度 |
| **A** Achievable | 可達成 | 有挑戰但不不切實際 |
| **R** Relevant | 相關性 | 與人生大方向一致 |
| **T** Time-bound | 有期限 | 三個月內達成 |

### 品牌元素

```markdown
## 個人品牌畫布

### 核心訊息
- **一句話定位**：我幫助 [誰] 解決 [什麼問題] 透過 [什麼方式]
- **關鍵詞**：3-5 個代表你的詞彙
- **差異化**：你和別人不同在哪裡？

### 內容策略
- **主題領域**：專注 2-3 個主題
- **內容形式**：文章/影片/Podcast/社群
- **發布頻率**：可持續的節奏

### 視覺識別
- **頭像**：專業且一致
- **色彩**：1-2 個主色
- **風格**：正式/友善/專業/活潑

### 平台選擇
| 平台 | 適合 | 內容類型 |
|------|------|----------|
| LinkedIn | B2B、專業 | 長文、見解 |
| Twitter/X | 科技、思想 | 短文、互動 |
| Instagram | 生活風格 | 圖片、短影片 |
| YouTube | 教學、分享 | 長影片 |
| 部落格 | SEO、深度 | 長文、教學 |
```

### 建立個人品牌步驟

```markdown
## 90 天個人品牌啟動計劃

### 第 1-30 天：定位
- [ ] 完成個人品牌畫布
- [ ] 決定 2-3 個專注主題
- [ ] 選擇 1-2 個主要平台
- [ ] 建立基本 Profile

### 第 31-60 天：內容
- [ ] 建立內容行事曆
- [ ] 產出 8-12 篇內容
- [ ] 找到自己的風格
- [ ] 開始互動和 networking

### 第 61-90 天：放大
- [ ] 分析什麼內容最有效
- [ ] 加倍投入高效內容
- [ ] 建立 email list
- [ ] 嘗試合作或曝光機會
```

## 時間管理

### 艾森豪矩陣

```
              緊急                 不緊急
         ┌─────────────────┬─────────────────┐
         │                 │                 │
    重   │   ⚡ 立即做      │   📅 排程做     │
    要   │   危機、截止日期 │   規劃、成長    │
         │   Do It         │   Schedule It   │
         │                 │                 │
         ├─────────────────┼─────────────────┤
         │                 │                 │
    不   │   👥 委派       │   🗑️ 刪除      │
    重   │   會議、郵件     │   瑣事、干擾    │
    要   │   Delegate It   │   Delete It     │
         │                 │                 │
         └─────────────────┴─────────────────┘

黃金法則：花更多時間在「重要但不緊急」象限
```

### 番茄工作法

```markdown
## Pomodoro Technique

1. **選擇任務** - 決定要做什麼
2. **設定 25 分鐘** - 專注工作
3. **休息 5 分鐘** - 離開螢幕
4. **每 4 個番茄** - 休息 15-30 分鐘

進階技巧：
- 追蹤每日番茄數
- 預估任務需要幾個番茄
- 記錄中斷次數並改善
```
