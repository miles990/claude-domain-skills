## Note-Taking Methods

### Zettelkasten Method

```markdown
## Principles

1. **Atomicity** - One concept per note
2. **Connectivity** - Notes link to each other
3. **Own Words** - Rewrite in your understanding
4. **Permanent Notes** - Processed, thoughtful notes

## Note Types

| Type | Description | Retention |
|------|-------------|-----------|
| Fleeting Notes | Quick capture of ideas | Temporary |
| Literature Notes | Excerpts while reading | Reference |
| Permanent Notes | Digested understanding | Permanent |
| Project Notes | Project-specific | Project duration |

## Note Template

Title: [Concept Name]
Date: YYYY-MM-DD
Source: [Origin]
Links: [[Related Note 1]] [[Related Note 2]]

---
[Explain the concept in your own words]

---
Why it matters:
How to apply:
```

## AI-Assisted Knowledge Management

```markdown
## AI Tool Integration

### Capture Phase
- Voice to text (Whisper, iOS Notes)
- Image OCR (Google Lens, iOS Live Text)
- Web summarization (ChatGPT, Claude)

### Organize Phase
- Auto tag suggestions
- Content classification
- Duplicate detection

### Distill Phase
- Long-form summarization
- Key concept extraction
- Q&A generation

### Express Phase
- Writing assistance
- Content rewriting
- Multi-language translation

## AI Prompt Examples

### Note Summarization
"Please organize this content into structured notes including:
1. Core points (3-5 items)
2. Key quotes
3. Action items
4. Connections to my existing knowledge"

### Concept Explanation
"Explain [concept] using the Feynman Technique:
1. Explain it simply to a 12-year-old
2. Identify gaps in my understanding
3. Use analogies to make it memorable"
```

## Daily/Weekly/Monthly/Quarterly Rhythms

```markdown
## Daily (5-10 minutes)
- Clear inbox (email, notes, tasks)
- Process fleeting notes
- Confirm today's priorities

## Weekly (30-60 minutes)
- [ ] Weekly review
- [ ] Organize this week's notes
- [ ] Plan next week's priorities
- [ ] Process "read later" backlog

## Monthly (1-2 hours)
- [ ] Monthly review
- [ ] Review project progress
- [ ] Organize and archive
- [ ] Update goals and priorities

## Quarterly (half day)
- [ ] Quarterly deep review
- [ ] Assess annual goal progress
- [ ] Adjust systems and processes
- [ ] Clean outdated content
- [ ] Plan next quarter focus
```

## Common Pitfalls

| Trap | Solution |
|------|----------|
| Collector's Fallacy | Only collect what you'll use |
| Tool Obsession | Tools serve purpose, not the goal |
| Over-categorization | Search > Classification |
| Input Without Output | Force output, learn by doing |
| Perfectionism | Starting > Perfect |

## Recommended Tools

- **Notes**: Obsidian, Notion, Logseq, Roam
- **Reading**: Readwise, Kindle, Pocket
- **Tasks**: Todoist, Things 3, TickTick
- **Automation**: Zapier, Make, Shortcuts
- **Writing**: Ulysses, iA Writer, Typora
