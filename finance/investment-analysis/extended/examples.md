## 杜邦分析 (DuPont Analysis)

### 完整框架圖

```
┌─────────────────────────────────────────────────────────────────┐
│  杜邦分析：拆解 ROE 的三個驅動因素                               │
│                                                                 │
│  ROE = 淨利率 × 資產週轉率 × 財務槓桿                          │
│                                                                 │
│      淨利       營收        總資產                              │
│  = ────── × ────── × ──────                                    │
│      營收      總資產      股東權益                              │
│                                                                 │
│  ┌────────────┬────────────┬────────────┐                      │
│  │   獲利能力 │   營運效率 │   財務結構 │                      │
│  │   淨利率   │   週轉率   │   槓桿倍數 │                      │
│  └────────────┴────────────┴────────────┘                      │
│                                                                 │
│  分析重點：                                                     │
│  • 哪個因素貢獻最多？                                           │
│  • 高 ROE 是靠獲利還是靠槓桿？                                  │
│  • 趨勢變化原因是什麼？                                         │
└─────────────────────────────────────────────────────────────────┘
```

### 杜邦分析範例比較

| 公司 | 淨利率 | 週轉率 | 槓桿 | ROE | 解讀 |
|------|--------|--------|------|-----|------|
| A 公司 | 20% | 0.8 | 1.5 | 24% | 高利潤型：靠毛利率高取勝 |
| B 公司 | 5% | 3.0 | 2.0 | 30% | 高週轉型：薄利多銷模式 |
| C 公司 | 8% | 1.2 | 3.0 | 28.8% | 高槓桿型：財務風險較高 |

**投資啟示**：
- A 公司：品牌或技術優勢，護城河可能較深
- B 公司：零售或快消行業特徵，需關注競爭
- C 公司：需警惕負債風險，了解槓桿來源

## 產業生命週期分析

### 完整生命週期圖

```
┌─────────────────────────────────────────────────────────────────┐
│  產業生命週期與投資策略                                          │
│                                                                 │
│  市場                                                           │
│  規模 ↑                                                         │
│       │              成熟期                                     │
│       │           ┌──────────┐                                 │
│       │          /            \                                 │
│       │   成長期/              \衰退期                          │
│       │        /                \                               │
│       │   導入期                  \                              │
│       │  /                         \                            │
│       └────────────────────────────────→ 時間                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 各階段特徵與策略

| 階段 | 特徵 | 風險 | 投資策略 |
|------|------|------|----------|
| **導入期** | 高成長潛力 | 技術風險、負現金流 | 創投型、高風險高報酬 |
| **成長期** | 快速增長、競爭加劇 | 市佔搶奪、資金需求 | 成長股（高 P/E 可接受） |
| **成熟期** | 穩定現金流、價格競爭 | 成長趨緩 | 價值股（穩定股息） |
| **衰退期** | 市場萎縮、整併機會 | 結構性衰退 | 避開或特殊情況投資 |

## 護城河評估清單

### 五種護城河類型

| 類型 | 說明 | 範例 |
|------|------|------|
| **品牌** | 消費者願意付溢價 | 可口可樂、愛馬仕 |
| **轉換成本** | 換產品代價高 | SAP、Adobe |
| **網路效應** | 用戶越多價值越高 | Facebook、Visa |
| **成本優勢** | 低成本生產 | Walmart、台積電 |
| **無形資產** | 專利、許可證 | 輝瑞、特許經營 |

### 品牌護城河評估
- [ ] 消費者能說出品牌名
- [ ] 願意付溢價購買
- [ ] 品牌忠誠度高
- [ ] 品牌價值持續多年

### 轉換成本護城河評估
- [ ] 客戶更換需要高額費用
- [ ] 需要重新培訓人員
- [ ] 數據遷移困難
- [ ] 合約綁定長期

### 網路效應護城河評估
- [ ] 用戶增加提升產品價值
- [ ] 存在雙邊市場效應
- [ ] 競爭者難以複製網路規模
- [ ] 用戶黏性高

### 護城河持久性評估
| 評級 | 描述 | 投資建議 |
|------|------|----------|
| 寬廣且持久 | 優勢明顯、難以複製 | 強力買入 |
| 寬廣但受威脅 | 有優勢但面臨挑戰 | 謹慎評估 |
| 狹窄 | 優勢較弱或易複製 | 短期機會 |
| 無護城河 | 無差異化 | 純價格競爭，避開 |

## 波特五力分析框架

```
┌─────────────────────────────────────────────────────────────────┐
│  波特五力分析 (Porter's Five Forces)                             │
│                                                                 │
│                 新進入者威脅                                     │
│                      ↓                                          │
│    供應商       →  產業競爭  ←      買方                        │
│    議價能力           ↑            議價能力                      │
│                 替代品威脅                                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 評估問題清單

| 力量 | 評估問題 | 高分情境（對產業有利） |
|------|----------|----------------------|
| 新進入者威脅 | 進入障礙高嗎？（資本、法規、技術） | 障礙高、進入難 |
| 供應商議價能力 | 供應商集中嗎？有議價權嗎？ | 供應商分散 |
| 買方議價能力 | 買方有選擇嗎？價格敏感嗎？ | 買方分散、品牌忠誠 |
| 替代品威脅 | 有替代品嗎？切換成本高嗎？ | 無替代或切換成本高 |
| 現有競爭 | 競爭激烈嗎？差異化程度？ | 差異化大、競爭有序 |

**評分方式**：每項 1-5 分，總分越高產業越有吸引力

## 情境估值法範例

### 三情境估值

```
┌─────────────────────────────────────────────────────────────────┐
│  三情境估值法                                                    │
│                                                                 │
│  樂觀情境 (Bull Case)                                           │
│  ├─ 假設：營收成長 15%，毛利率提升                              │
│  ├─ 機率：25%                                                   │
│  └─ 估值：$150                                                  │
│                                                                 │
│  基準情境 (Base Case)                                           │
│  ├─ 假設：營收成長 8%，維持現狀                                 │
│  ├─ 機率：50%                                                   │
│  └─ 估值：$120                                                  │
│                                                                 │
│  悲觀情境 (Bear Case)                                           │
│  ├─ 假設：營收持平，利潤下滑                                    │
│  ├─ 機率：25%                                                   │
│  └─ 估值：$80                                                   │
│                                                                 │
│  加權估值 = 0.25×150 + 0.50×120 + 0.25×80 = $117.50            │
│                                                                 │
│  決策：若目前股價 < $95 (20% 安全邊際)，考慮買入               │
└─────────────────────────────────────────────────────────────────┘
```

## 資產配置模型

### 風險承受度 vs 資產配置

| 類型 | 股票 | 債券 | 現金 | 適合對象 |
|------|------|------|------|----------|
| 保守型 | 30% | 50% | 20% | 退休者、低風險承受 |
| 穩健型 | 50% | 35% | 15% | 中年、穩定收入 |
| 積極型 | 70% | 20% | 10% | 年輕、長期投資 |

### 配置原則
- **年齡法則**：債券比例 = 100 - 年齡
- **分散原則**：單一持股不超過 10%
- **再平衡**：每年檢視調整
